# Swiggy Sales Data Analytics Dashboard
## Project Overview
This project focuses on analyzing Swiggy food delivery sales data and creating an interactive Excel dashboard to understand business performance, sales trends, customer ratings, and regional insights.
The dashboard provides key metrics and visual analysis to support data-driven decision making.
---
## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Dashboard Design
---
## Key Performance Indicators (KPIs)
The dashboard tracks:
- Total Sales
- Total Orders
- Average Order Value
- Average Customer Rating
- Rating Count
---
## Dashboard Features
### Sales Analysis
- Monthly Sales Trend
- Daily Sales Analysis
- Quarter-wise Sales Comparison
### Customer Analysis
- Average Rating Analysis
- Rating Count
### Category Analysis
- Veg vs Non-Veg Sales Comparison
- Food Category Performance
### Geographic Analysis
- Top Cities by Sales
- Regional Sales Performance
---
## Interactive Filters
The dashboard includes slicers for:
- Month
- State
- Food Type
- Category
